从 1.9.0 开始，MarkInside 提供了本地 **MCP（Model Context Protocol）服务器**。通过 MCP 将 MarkInside 连接到 Codex、Claude Code 等 AI 智能体，您可以通过自然语言让智能体直接编辑 MarkInside 中的内容，辅助您生成图片。

比如说您想生成一张正态分布的公式图，您可以直接对智能体说：“帮我在 MarkInside 中生成一张正态分布的公式图”，智能体就会在 MarkInside 中创建一个新的文档，并自动输入对应的公式。您也可以持续地要求智能体修改图片，直到您满意为止。

## 直接连接 MCP 服务器

默认情况下，服务器监听端口为 **9448**。

您可以在 **MarkInside → Settings → MCP Server** 中查看或修改端口。

将下面的 `<PORT>` 替换成您的实际端口号（如果没有修改过，使用 `9448`）。

<Tabs syncKey="agent">
<TabItem label="Codex">

先通过 Codex 命令添加到全局作用域：

```bash
codex mcp add --transport http markinside http://127.0.0.1:<PORT>/mcp
```

也可以直接写入配置文件：

- 用户级：`~/.codex/config.toml`
- 项目级：`.codex/config.toml`

```toml
[mcp_servers.markinside]
url = "http://127.0.0.1:<PORT>/mcp"
```

</TabItem>
<TabItem label="Claude Code">

注册 MCP 服务器：

```bash
claude mcp add --transport http markinside http://127.0.0.1:<PORT>/mcp
```

如果要注册到项目作用域，运行：

```bash
claude mcp add --transport http --scope project markinside http://127.0.0.1:<PORT>/mcp
```

也可以修改项目配置文件 `.mcp.json`：

```json
{
  "mcpServers": {
    "markinside": {
      "url": "http://127.0.0.1:<PORT>/mcp"
    }
  }
}
```

</TabItem>
</Tabs>
